# Frequently Asked Questions

### What Is The TLOS Contract Address?

#### Telos Zero
TLOS Contract address on Telos Zero: `eosio.token` ([Explorer](https://explorer.telos.net/account/eosio.token))

#### Telos EVM
TLOS on the TelosEVM is the gas token, it does not have a contract address

The standard WTLOS contract address on the Telos EVM is:
`0xD102cE6A4dB07D247fcc28F366A623Df0938CA9E` ([Explorer](https://teloscan.io/address/0xD102cE6A4dB07D247fcc28F366A623Df0938CA9E))

#### Bridged TLOS
TLOS is bridged via LayerZero OFT token standard from TelosEVM to many other networks:
- Ethereum `0x193f4a4a6ea24102f49b931deeeb931f6e32405d` ([Explorer](https://etherscan.io/address/0x193f4a4a6ea24102f49b931deeeb931f6e32405d))
- Base `0x7252c865c05378Ffc15120F428dd65804dD0CE63` ([Explorer](https://basescan.org/address/0x7252c865c05378Ffc15120F428dd65804dD0CE63))
- BNB `0x193f4a4a6ea24102f49b931deeeb931f6e32405d` ([Explorer](https://bscscan.com/address/0x193f4a4a6ea24102f49b931deeeb931f6e32405d))
- Avalanche `0xed667dc80a45b77305cc395db56d997597dc6ddd` ([Explorer](https://avascan.info/blockchain/c/token/0xed667dC80a45b77305Cc395DB56D997597Dc6DdD))
