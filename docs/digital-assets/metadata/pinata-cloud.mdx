# Pi単ata.cloud

## Step I - Create An Account

To get started head to [Pinata](https://www.pinata.cloud/)

1. Once you're on the home page click on the "Start Building" button.

<img
  src={require('/static/img/pinata-1.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

2. Register for a new account by submitting all information requested.
3. Confirm your new account with the confirmation code sent to your email and you'll then be taken to your account dashboard.

## Step II - Uploading Your Files

Pi単ata allows you to upload a single file or a folder containing multiple files. In terms of Digital Assets uploaded folders would most likely contain your art (pngs) or your metadata (json).

1. Click "Add Files" at the top right.

<img
  src={require('/static/img/pinata-2.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

2. Select the file or folder you'd like to upload. In this case we will be uploading a folder.

<img
  src={require('/static/img/pinata-3.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

:::info
Here is an example of what you would want your folder to look like.
This is a folder containing the png files for our Digital Assets. You'll want to upload these first so that you can put their newly created IPFS links into their respective metadata files.
:::

<img
  src={require('/static/img/pinata-4.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

3. Once your folder is uploaded you should be able to click on it to access its IPFS Link. This may take some time to load if you're uploading a larger amount of files.
4. Copy your folders CID number which is highlighted in blue.

<img
  src={require('/static/img/pinata-5.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

5. Take the CID along with the png name and place them into the appropriate metadata (json) files. 

:::info
You can find the Metadata folder we used for this example below
:::

<img
  src={require('/static/img/pinata-6.png').default}
  alt="pinata"
  width="70%"
  height="automatic"
/>

:::warning
These json files are used by your Digital Asset contract, dApps and marketplaces to pull all needed information about your Digital Asset. Be sure to double check that each file points to the correct png and has the right data.
:::

In the image above you can see that I placed the CID and the image name for the  first Digital Asset into the First json file. 

```JSON
"image": "ipfs://"CID"/filename.png"
```

6. Once your json files have been updated appropriately you can then upload the folder to pi単ata.

  alt="pinata"
  width="70%"
  height="automatic"
/>
